libsavestate::impl_save_state! [−] [src]

macro_rules! impl_save_state {
    ( $t:ident { $( $field:ident ),* } ignore { $( $ignore:ident ),* } ) => { ... };
}

Generates an impl of SaveState for a given type, saving/restoring a list of fields, and ignoring a second list of fields.

To prevent bugs, these lists must contains all fields of the type exactly once or this won't compile.

Example

#[macro_use]
extern crate libsavestate;

struct Struct {
    field: u8,
    field2: Vec<u8>,
    ignored: String,
}

impl_save_state!(Struct {
    field, field2
} ignore {
    ignored
});
