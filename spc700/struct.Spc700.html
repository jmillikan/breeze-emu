The SPC700 is an 8-bit processor with a 16-bit address space.

It has 64 KB of RAM shared with the DSP. The last 64 Bytes in its address space are mapped to the "IPL ROM", which contains a small piece of startup code that allows the main CPU to transfer a program to the APU.
